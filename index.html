My personal website

This is the very site you are on! I picked a more traditional route this time with HTML, SASS, vanilla JavaScript and Vite. HTML is tried and tested and I did not feel the need for React. Sass is incredibly helpful and had all the features I need for this project. I picked Vite as a bundler because of the little overhead it has and how fast it is, it was a real pleasure to use.
