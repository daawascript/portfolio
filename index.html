Weather app

A simple weather app built with React and Tailwind css, using the open weather api and Firebase for serverless functions. I picked those tools as React is great at creating modular components while keeping the source code maintanable and scalable. As for Tailwind, it is a very easy way to style applications without managing complex and hard to maintain stylesheets. Firebase helped me to make API calls while keeping my API key secure.

My personal website
